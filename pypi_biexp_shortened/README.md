# biexp-decay-sim

Simulating bi-exponential fluorescence decay curves and exporting
them as 3D TIFF datasets.

## Requirements

* Python >=3.8
* NumPy
* Tifffile

## Installation
```bash
pip install biexp-decay-sim
```

or

1. Clone the repository:

   ```bash
   git clone https://github.com/hwilson23/FLIM_varability_simulation.git
   cd FLIM_varability_simulation
   ```

2. Install the required packages:

   ```bash
   pip install -r requirements.txt
   ```
